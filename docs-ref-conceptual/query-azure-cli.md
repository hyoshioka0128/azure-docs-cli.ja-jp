<span data-ttu-id="b98bb-160">前のセクションでは、配列をフラット化して、リソース グループ内のすべての VM の完全なリストを取得しました。</span><span class="sxs-lookup"><span data-stu-id="b98bb-160">In the last section, we flattened an array to get the complete list of all VMs in a resource group.</span></span> <span data-ttu-id="b98bb-161">フィルターを使用すると、この出力を Linux VM だけに制限できます。</span><span class="sxs-lookup"><span data-stu-id="b98bb-161">Using filters, this output can be restricted to only Linux VMs:</span></span>
